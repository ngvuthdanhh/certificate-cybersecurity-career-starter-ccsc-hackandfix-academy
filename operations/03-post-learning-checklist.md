## 2. Completion Checklist

[ ] Hypothesis written per topic
[ ] Lab report produced (150-200 words)

## 3. Required Outputs
| Output | Requirement |
|---|---|
| **Hypothesis Doc** | 60-100 words |
| **Lab Report** | 150-200 words |
| **Portfolio Entry** | 1-line index description |
| **Confidence Rating** | 1-5 based on evidence quality |

---

## 4. Confidence Rating Rubric
| Score | Meaning |
|---|---|
| 5 | Strong correlation + high source credibility |
| 4 | Reliable evidence but minor gaps |
| 3 | Plausible but needs more validation |
| 2 | Weak correlation or limited evidence |
| 1 | Not reliable, must be re-analyzed |
